- **移除功能**: 不再将 MarketData 存入数据库

#### 2.2 实现自动清理机制
- **清理策略**: 自动删除超过24小时的点差记录
- **执行时机**: 每次存储新点差记录时自动清理
- **实现方式**: 使用 `timedelta(days=1)` 计算截止时间
- **日志记录**: 记录每次清理的数据条数

#### 2.3 更新数据查询逻辑
- 修改 `get_latest_market_data()` 方法
- 从 `spread_records` 表读取最新数据

# 声音文件管理功能说明

## 功能概述

系统提供了完整的声音文件管理功能，支持：
1. 上传声音文件到本地服务器
2. 自动同步声音文件到飞书云文档
3. 在线预听声音文件
4. 为通知模板配置声音提醒

## 使用方法

### 1. 访问系统管理页面

访问: http://13.115.21.77:3000/system

### 2. 配置通知模板声音

1. 在"通知模板管理"区域找到需要配置的模板
2. 点击"声音设置"按钮
3. 在弹出的对话框中：
   - 选择已有的声音文件
   - 或点击"选择文件"上传新的声音文件
   - 设置重复次数（1-10次）
   - 点击"试听声音"预览效果
4. 点击"保存"完成配置

### 3. 上传声音文件

**方式一：通过界面上传**
1. 在声音设置对话框中点击"选择文件"
2. 选择音频文件（支持 MP3, WAV, OGG, OPUS 格式）
3. 文件会自动上传到本地服务器并同步到飞书云文档

**方式二：手动上传**
1. 将音频文件放到服务器的 `frontend/public/sounds/` 目录
2. 在声音设置对话框中点击"同步"按钮（刷新图标）
3. 系统会将所有本地文件同步到飞书云文档

### 4. 同步到飞书云文档

点击声音设置对话框中的"同步"按钮（刷新图标），系统会：
1. 扫描本地 `frontend/public/sounds/` 目录
2. 将所有音频文件上传到飞书云文档
3. 显示同步结果（成功/失败数量）

## API 接口

### 获取声音文件列表
```
GET /api/v1/sounds
```

### 上传声音文件
```
POST /api/v1/sounds/upload
Content-Type: multipart/form-data

file: <音频文件>
```

### 同步到飞书
```
POST /api/v1/sounds/sync-to-feishu
```

### 删除声音文件
```
DELETE /api/v1/sounds/{filename}
```

## 文件存储

- **本地存储**: `frontend/public/sounds/`
- **访问路径**: `http://13.115.21.77:8000/sounds/{filename}`
- **飞书云文档**: 通过飞书 API 自动上传

## 支持的音频格式

- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- OPUS (.opus)

## 注意事项

1. **系统级设置**: 声音配置对所有用户生效
2. **文件大小**: 建议单个文件不超过 10MB
3. **飞书限制**: 飞书 API 对文件大小和格式有限制
4. **自动同步**: 上传文件时会自动同步到飞书，无需手动操作
5. **预听功能**: 保存前可以试听声音效果

## 故障排查

### 上传失败
- 检查文件格式是否支持
- 检查文件大小是否过大
- 检查服务器磁盘空间

### 同步到飞书失败
- 检查飞书服务配置是否正确
- 检查网络连接
- 查看后端日志获取详细错误信息

### 无法预听
- 检查浏览器是否支持该音频格式
- 检查文件路径是否正确
- 检查文件是否损坏

## 技术实现

- **后端**: FastAPI + aiohttp
- **前端**: Vue 3 + Composition API
- **存储**: 本地文件系统 + 飞书云文档
- **API**: 飞书开放平台 API

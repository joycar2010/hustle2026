# 交易系统界面重构文档

## 概述
基于专业交易平台设计的全新套利交易系统界面，采用深色模式，高信息密度布局，适合专业交易员使用。

## 组件说明

### 1. 左侧边栏

#### AccountStatusPanel.vue - 账户状态面板
**功能**: 实时显示账户核心指标
**数据项**:
- 账户总资产 (带24h涨跌幅)
- 可用总资产
- 净资产
- 总持仓数量
- 冻结资产
- 当日盈亏 (带ROI百分比)
- 保证金余额
- 风险率 (带可视化进度条)
- 资金费/掉期费 (Bybit/Binance)

**更新频率**: 5秒

#### NavigationPanel.vue - 功能导航面板
**功能**: 快速切换功能模块
**导航项**:
- 账户管理
- 策略控制 (带运行状态徽章)
- 风控状态 (带风险等级徽章)

### 2. 主体区域 - 顶部

#### SpreadChart.vue - 点差走势图
**功能**: 实时点差趋势可视化
**特性**:
- Chart.js 实现的流畅折线图
- 多时间周期切换 (15m, 1H, 4H, 24H)
- 实时统计数据 (当前/均值/最高/最低)
- 自动更新 (2秒间隔)

#### MarketCards.vue - 行情卡片
**功能**: 双交易所实时报价
**显示内容**:
- Binance 买价/卖价/点差
- Bybit 买价/卖价/点差
- 连接状态指示器
- 套利机会提示 (当点差>0.5时)

**价格变动视觉反馈**:
- 上涨: 绿色 (#0ecb81)
- 下跌: 红色 (#f6465d)
- 不变: 白色

#### SpreadDataTable.vue - 点差数据流水表
**功能**: 实时点差数据流
**列**: 时间 | 点差 | 方向
**特性**:
- 自动滚动更新
- 颜色编码 (点差>3绿色, >2黄色)
- 最多显示50条记录

### 3. 主体区域 - 中部

#### StrategyPanel.vue - 策略配置面板
**功能**: 正向/反向套利策略配置
**配置项**:
- 最小点差
- 最大点差
- 单次数量
- 最大持仓
- 止损点差

**阶梯配置**:
- 动态添加/删除阶梯
- 每个阶梯包含: 点差阈值 + 交易数量

**操作按钮**:
- 保存配置 (黄色 #f0b90b)
- 启动/停止策略 (绿色/灰色切换)
- 平仓 (红色 #f6465d, 仅运行时显示)

### 4. 主体区域 - 底部

#### OrderMonitor.vue - 挂单监控
**功能**: 实时挂单状态监控
**列**: 时间 | 交易所 | 方向 | 数量 | 价格 | 状态
**状态颜色**:
- 挂单中: 黄色
- 已成交: 绿色
- 已取消: 红色

#### RiskDashboard.vue - 风险仪表盘
**功能**: 实时风险监控
**指标**:
- 保证金使用率 (带可视化进度条)
- 最大回撤
- 胜率
- 盈亏比
- 风险预警列表

**风险等级**:
- <30%: 低风险 (绿色)
- 30-60%: 中风险 (黄色)
- >60%: 高风险 (红色)

#### ManualTrading.vue - 手动交易面板
**功能**: 手动下单执行
**输入项**:
- 交易所选择 (Binance/Bybit)
- 交易对
- 数量
- 价格 (留空为市价)

**操作按钮**:
- 买入 (绿色)
- 卖出 (红色)
- 平仓所有持仓 (红色)
- 取消所有挂单 (灰色)

## 交互规范

### 状态反馈
1. **启用状态**: 绿色高亮 + "运行中"文字
2. **停止状态**: 灰色 + "已停止"文字
3. **保存操作**: 黄色按钮
4. **危险操作**: 红色按钮 + 确认对话框

### 数据更新
- 价格数据: 1秒更新
- 点差图表: 2秒更新
- 账户数据: 5秒更新
- 风险数据: 10秒更新

### 视觉反馈
- 价格变动: 颜色闪烁 (绿/红)
- 按钮悬停: 颜色加深
- 输入框聚焦: 黄色边框
- 表格行悬停: 背景变深

- **框架**: Vue 3 (Composition API)
- **样式**: TailwindCSS
- **图表**: Chart.js
- **状态管理**: Pinia
- **路由**: Vue Router

## API 集成

所有组件已预留API调用接口，需要连接后端API:

```javascript
// 示例: 获取账户数据
async function fetchAccountData() {
  const response = await api.get('/api/v1/accounts/summary')
  accountData.value = response.data
}

// 示例: 保存策略配置
async function saveConfig() {
  await api.post('/api/v1/strategies/config', config.value)
}

// 示例: 执行交易
async function executeTrade(side) {
  await api.post('/api/v1/orders/create', {
    exchange: tradeForm.value.exchange,
    symbol: tradeForm.value.symbol,
    side,
    quantity: tradeForm.value.quantity,
    price: tradeForm.value.price
  })
}
```

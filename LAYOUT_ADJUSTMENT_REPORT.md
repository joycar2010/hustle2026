## 一、TradingDashboard.vue 布局调整

### 底部区域宽度重新分配

**之前**：
- OrderMonitor: `flex-1` (自适应，占据大部分空间)
- MarketCards: `w-80` (320px 固定宽度)

**现在**：
- OrderMonitor: `w-[35%]` (35% 宽度，减少 50%)
- MarketCards: `flex-1` (自适应，增加 50%+)

```vue
<!-- 之前 -->
<section class="flex-1 flex gap-2 p-2 min-h-0">
  <div class="flex-1 bg-[#1e2329] rounded overflow-hidden">
    <OrderMonitor />
  </div>
  <div class="w-80 bg-[#1e2329] rounded overflow-hidden flex-shrink-0">
    <MarketCards />
  </div>
</section>

<!-- 现在 -->
<section class="flex-1 flex gap-2 p-2 min-h-0">
  <div class="w-[35%] bg-[#1e2329] rounded overflow-hidden flex-shrink-0">
    <OrderMonitor />
  </div>
  <div class="flex-1 bg-[#1e2329] rounded overflow-hidden">
    <MarketCards />
  </div>
</section>
```


### 3. 布局改为双列

**之前**：所有输入框单列排列
```vue
<div class="space-y-2">
  <div>
    <label>Binance 净资产提醒值 (USDT)</label>
    <input ... />
  </div>
  <div>
    <label>Bybit MT5 净资产提醒值 (USDT)</label>
    <input ... />
  </div>
  <div>
    <label>总资产提醒值 (USDT)</label>
    <input ... />
  </div>
</div>
```

**现在**：使用 Grid 双列布局
```vue
<div class="grid grid-cols-2 gap-2">
  <div>
    <label class="block text-[10px] mb-1">Binance 净资</label>
    <input class="w-full px-2 py-1 text-xs" ... />
  </div>
  <div>
    <label class="block text-[10px] mb-1">Bybit 净资</label>
    <input class="w-full px-2 py-1 text-xs" ... />
  </div>
  <div class="col-span-2">
    <label class="block text-[10px] mb-1">总资产</label>
    <input class="w-full px-2 py-1 text-xs" ... />
  </div>
</div>
```

### 4. 字体和间距优化

| 元素 | 之前 | 现在 | 说明 |
|------|------|------|------|
| 卡片内边距 | p-3 | p-2 | 减少间距 |
| 标题字体 | text-xs | text-xs | 保持不变 |
| 标签字体 | text-xs | text-[10px] | 缩小字体 |
| 输入框字体 | text-sm | text-xs | 缩小字体 |
| 输入框内边距 | py-1.5 | py-1 | 减少高度 |
| 风险指标数值 | text-xl | text-base | 缩小字体 |

## 四、主要改进总结

### TradingDashboard.vue
1. ✅ OrderMonitor 宽度从 flex-1 减少到 35%
2. ✅ MarketCards 宽度从 320px 增加到 flex-1 (约 65%)
3. ✅ 点差数据流显示区域更紧凑
4. ✅ 行情卡片显示区域更宽敞

### Risk.vue
1. ✅ 风险指标改为单行显示，节省垂直空间
2. ✅ 所有字段名称大幅简化，提升可读性
3. ✅ 表单布局从单列改为双列，充分利用 320px 宽度
4. ✅ 字体和间距全面优化，更加紧凑
5. ✅ 标题更简洁明了（净资产提醒、爆仓价位提醒、MT5卡顿、反向提醒、正向提醒）
6. ✅ 字段命名更准确（点差值、同步条数）

### 用户体验提升
1. **空间利用率提升**：双列布局充分利用 320px 宽度
2. **信息密度优化**：单行显示风险指标，减少滚动
3. **命名更简洁**：去除冗余文字，提升扫描效率
4. **视觉更清晰**：Grid 布局对齐整齐，易于阅读

# Hustle XAU - Design System Specification

## Color Scheme

### Primary Colors
- **Primary Gold**: `#F0B90B` - Main brand color, CTAs, highlights
- **Primary Gold Hover**: `#E0A800` - Hover states
- **Primary Gold Light**: `#FFF8E1` - Light backgrounds

### Background Colors
- **Dark 300 (Main BG)**: `#0B0E11` - Main background
- **Dark 200 (Card BG)**: `#181A20` - Card backgrounds
- **Dark 100 (Elevated)**: `#1E2329` - Elevated elements, navbar
- **Dark 50 (Hover)**: `#2B3139` - Hover states

### Semantic Colors
- **Success Green**: `#0ECB81` - Positive values, buy actions
- **Success Green Dark**: `#0DB774` - Hover states
- **Danger Red**: `#F6465D` - Negative values, sell actions
- **Danger Red Dark**: `#E03D52` - Hover states
- **Warning Orange**: `#FF9800` - Warnings, alerts
- **Info Blue**: `#2196F3` - Information, neutral actions

### Text Colors
- **Text Primary**: `#FFFFFF` - Main text
- **Text Secondary**: `#B7BDC6` - Secondary text
- **Text Tertiary**: `#848E9C` - Tertiary text, labels
- **Text Disabled**: `#5E6673` - Disabled text

### Border Colors
- **Border Primary**: `#2B3139` - Main borders
- **Border Secondary**: `#1E2329` - Subtle borders
- **Border Focus**: `#F0B90B` - Focus states

## Typography

### Font Family
- **Primary**: `'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
- **Monospace**: `'Roboto Mono', 'Courier New', monospace` - For numbers, prices

### Font Sizes
- **xs**: `0.75rem` (12px)
- **sm**: `0.875rem` (14px)
- **base**: `1rem` (16px)
- **lg**: `1.125rem` (18px)
- **xl**: `1.25rem` (20px)
- **2xl**: `1.5rem` (24px)
- **3xl**: `1.875rem` (30px)
- **4xl**: `2.25rem` (36px)

### Font Weights
- **normal**: 400
- **medium**: 500
- **semibold**: 600
- **bold**: 700

## Spacing Scale
- **0**: 0
- **1**: 0.25rem (4px)

- **3**: 0.75rem (12px)

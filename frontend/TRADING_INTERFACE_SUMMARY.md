## 🚀 如何使用

### 访问新界面
1. 登录系统: http://localhost:3000/login
2. 使用凭证: admin / admin123
3. 自动跳转到新交易界面

### 功能说明
1. **左侧边栏**: 查看账户状态和切换功能
2. **顶部区域**: 监控实时价格和点差
3. **中部区域**: 配置和控制套利策略
4. **底部区域**: 监控订单、风险和手动交易

### 操作流程
1. 查看账户状态和风险指标
2. 监控实时价格和点差走势
3. 配置正向/反向套利策略参数
4. 启动策略或手动执行交易
5. 监控挂单状态和风险预警

## 🔧 技术特性

- **响应式设计**: 自适应不同屏幕尺寸
- **实时更新**: 1-10秒不同频率更新
- **视觉反馈**: 价格变动、状态变化即时反馈
- **模块化**: 组件独立，易于维护
- **可扩展**: 预留API接口，易于集成

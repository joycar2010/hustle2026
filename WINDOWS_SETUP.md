# Windows Setup Guide - Hustle XAU

## Prerequisites Installation

### 1. Install PostgreSQL

**Download:**
- Visit: https://www.postgresql.org/download/windows/
- Download PostgreSQL 15 or later
- Run the installer

**During Installation:**
- Remember the password you set for the `postgres` user
- Keep default port: 5432
- Install pgAdmin (optional but recommended)

**After Installation:**
1. Open pgAdmin or use psql command line
2. Create the database:

```sql
-- Connect as postgres user
CREATE DATABASE hustle_db;
CREATE USER hustle_user WITH PASSWORD 'hustle_password';
GRANT ALL PRIVILEGES ON DATABASE hustle_db TO hustle_user;
ALTER DATABASE hustle_db OWNER TO hustle_user;
```

**Or use command line:**
```bash
# Add PostgreSQL to PATH first, then:
psql -U postgres -c "CREATE DATABASE hustle_db;"
psql -U postgres -c "CREATE USER hustle_user WITH PASSWORD 'hustle_password';"
psql -U postgres -c "GRANT ALL PRIVILEGES ON DATABASE hustle_db TO hustle_user;"
```

### 2. Install Redis

**Option A: Using Windows Subsystem for Linux (WSL)**
```bash
wsl --install
# After WSL is installed:
wsl
sudo apt update
sudo apt install redis-server
sudo service redis-server start
```

**Option B: Using Memurai (Redis for Windows)**
- Visit: https://www.memurai.com/
- Download Memurai (free for development)
- Install and start the service

**Option C: Using Docker**
```bash
docker run -d -p 6379:6379 redis:latest
```

**Verify Redis:**
```bash
redis-cli ping
# Should return: PONG
```

### 3. Verify Python

```bash
python --version
# Should show Python 3.9 or later
```

If not installed:
- Visit: https://www.python.org/downloads/
- Download Python 3.11 or later
- **Important:** Check "Add Python to PATH" during installation

### 4. Verify Node.js

```bash
node --version
npm --version
```

If not installed:
- Visit: https://nodejs.org/
- Download LTS version
- Run installer

## Backend Setup

### 1. Check Prerequisites

```bash
cd d:\git\hustle2026
python check_prerequisites.py
```

This will verify all required services are running.

### 2. Install Python Dependencies

```bash
cd backend
pip install -r requirements.txt
```

**If you get SSL errors:**
```bash
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org -r requirements.txt
```

### 3. Configure Environment

The `.env` file has been created with these settings:

```env
DATABASE_URL=postgresql://hustle_user:hustle_password@localhost:5432/hustle_db
REDIS_URL=redis://localhost:6379/0
SECRET_KEY=<auto-generated>
```

**If you used different credentials**, edit `backend\.env`:
```bash
notepad backend\.env
```

### 4. Run Database Migrations

```bash
cd backend
alembic upgrade head
```

**Expected output:**
```
INFO  [alembic.runtime.migration] Context impl PostgresqlImpl.
INFO  [alembic.runtime.migration] Will assume transactional DDL.
INFO  [alembic.runtime.migration] Running upgrade  -> abc123, Initial migration
```

**If you get errors:**

**Error: "database does not exist"**
```bash
# Create the database first (see step 1)
```

**Error: "password authentication failed"**
```bash
# Check your PostgreSQL password in .env file
# Make sure it matches what you set during PostgreSQL installation
```

**Error: "could not connect to server"**
```bash
# Make sure PostgreSQL is running
# Check Windows Services: services.msc
# Look for "postgresql-x64-15" and start it
```

### 5. Initialize Platform Data

```bash
python -m app.scripts.init_platforms
```

### 6. Create Test User

```bash
python create_test_user.py
```

**Expected output:**
```
✓ Test user created successfully!
  Username: admin
  Password: admin123
  Email: admin@hustle.com
  User ID: 1
```

### 7. Start Backend

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**Expected output:**
```
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
INFO:     Started reloader process
INFO:     Started server process
INFO:     Waiting for application startup.
INFO:     Application startup complete.
```

**Verify backend:**
- Open browser: http://localhost:8000
- Should see: `{"message": "Hustle XAU Arbitrage System API", ...}`
- API Docs: http://localhost:8000/docs

## Troubleshooting

### PostgreSQL Issues

**Service not starting:**
1. Open Services (Win + R, type `services.msc`)
2. Find "postgresql-x64-15"
3. Right-click → Start

**Can't connect:**
```bash
# Check if PostgreSQL is listening
netstat -an | findstr 5432
```

**Password issues:**
```bash
# Reset postgres password
# 1. Edit pg_hba.conf (usually in C:\Program Files\PostgreSQL\15\data\)
# 2. Change "md5" to "trust" for local connections
# 3. Restart PostgreSQL service
# 4. Connect and change password:
psql -U postgres
ALTER USER postgres PASSWORD 'newpassword';
# 5. Change pg_hba.conf back to "md5"
# 6. Restart PostgreSQL
```

### Redis Issues

**Redis not running:**
```bash
# If using Memurai:
# Open Services, start "Memurai" service

# If using WSL:
wsl
sudo service redis-server start

# If using Docker:
docker start <redis-container-id>
```

**Test Redis connection:**
```bash
redis-cli ping
# Should return: PONG
```

### Backend Issues

**Port 8000 already in use:**
```bash
# Find process using port 8000
netstat -ano | findstr :8000

# Kill the process
taskkill /PID <PID> /F
```

**Import errors:**
```bash
# Make sure you're in the backend directory
cd backend

# Reinstall dependencies
pip install -r requirements.txt
```

**Database connection errors:**
```bash
# Test database connection
python -c "from app.core.database import engine; print('Connected!')"
```

### Frontend Issues

**Port 3000 already in use:**
```bash
# Find and kill process
netstat -ano | findstr :3000
taskkill /PID <PID> /F
```

**API connection errors:**
- Make sure backend is running on port 8000
- Check browser console for CORS errors
- Verify `VITE_API_BASE_URL` in frontend\.env

**Build errors:**
```bash
# Clear node_modules and reinstall
rmdir /s /q node_modules
del package-lock.json
npm install
```
